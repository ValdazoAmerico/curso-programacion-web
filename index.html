CSS


CSS es un acrónimo de Cascading Style Sheets, es un lenguaje de estilos utilizado para describir la presentación de documentos HTML o XML.


CSS es uno de los lenguajes base de la Open Web y posee una especificación estandarizada por parte del W3C. Anteriormente , el desarrollo de varias partes de las especificaciones de CSS era realizado de manera sincrónica, lo que permiía el versionado de las recomendaciones. Probablemente habrás escuchado acerca de CSS1, CSS2.1, CSS3. Sin embargo, CSS4 nunca se ha lanzado como una versión oficial. Desde CSS3, el alcance de las especificaciones se incrementó de forma significativa y el progreso de los diferentes modulos de CSS comenzó a mostrar varias diferencias, lo que hizo más efectivo desarrollar y publicar recomendaciones separadas por módulos. En vez de versionar las especificaciones de CSS, la W3C actualmente realiza una captura de las ultimas espcificaciones estables de CSS.

Historia

1994:

CSS fue propuesto por primera vez por Håkon Wium Lie el 10 de octubre de 1994. Al mismo tiempo, Lie trabajaba con Tim Berners-Lee en el CERN.​ Muchos otros lenguajes de hojas de estilos fueron propuestos al mismo tiempo.

1996:

La fecha de lanzamiento inicial oficial fue el 17 de diciembre de 1996. Se dió lugar a la primera recomendación CSS por el W3C (CSS1). En particular, la propuesta de Bert Bos fue influyente; él fue el coautor de CSS1 y es reconocido como el cocreador de CSS.

1997:

A principios de 1997, el W3C decide separar los trabajos del grupo de HTML en tres secciones: el grupo de trabajo de HTML, el grupo de trabajo de DOM y el grupo de trabajo de CSS.

1999:

Los trabajos en el CSS3.1, comenzaron a la vez que se publicó la recomendación oficial de CSS2, y los primeros borradores de CSS3.1 fueron liberados en junio de 1999

2000:

El primer navegador con soporte completo de CSS 1 fue la versión para Mac de Internet Explorer 5, que se publicó en el año 2000.
